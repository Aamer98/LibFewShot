# 介绍
`LibFewShot`是一个主要用于小样本学习的综合算法库。其中以统一框架集成了多种经典的小样本学习方法，包含四种基于微调的方法，六种基于元学习的方法，以及八种基于度量学习的方法。本库代码简洁、结构明了，易于新接触小样本学习的新手学习使用。
