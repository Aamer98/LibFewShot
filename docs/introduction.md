# 介绍
`LibFewShot`是一个小样本学习算法库，其中集成
了近年来较为经典的小样本学习算法，包括基于元学习的、基于度量的以及基于微调的小样本算法，涵盖了基于全局特征的和基于局部特征的经典方法。本库面向小样本学习初学者，代码简洁、结构明了，易于新接触深度学习和小样本学习的新手学习使用。
